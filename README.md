# Basic-Linux-commands
Here is a list of 40 basic Linux commands


# Here is a list of 40 basic Linux commands:
   
   > List the contents of a directory
```shell     
   ls 
```
   
   > Change the working directory
```shell     
cd 
```
   
   >  Print the current working directory
   
   ```shell     

    pwd  

    ```
    
   >  Create a new directory
   
   ```shell     

    mkdir  

    ```
   
   >  Remove an existing directory
   
   ```shell     

    rmdir  

    ```
   
    >  Copy a file or directory
   
   ```shell     
   
    cp 
   
    ```
   
    >  Move or rename a file or directory
   
   ```shell     
   
    mv 
   
    ```
    
     >  Remove a file or directory
```shell     
rm 
```
    
    >  View the contents of a file
    
    ```shell     

    cat  

    ```

    >  View the contents of a file one page at a time
    
    ```shell     

    less  
    
    ```
    
    >  Search for a pattern in one or more files
    
    ```shell     
    
    grep  
    
    ```
    
    >  Search for files in a directory hierarchy
    
    ```shell     
    
    find  
    
    ```
    
    >  Sort lines of text files
    
    ```shell     
    
    sort  
    
    ```
    
    >  Print information about users currently logged in
    
    ```shell
         
    who  
    
    ```
    
    >  Print the current userid
    
    ```shell
         
    whoami  
    
    ```
    
    >  Print information about users
    
    ```shell
         
    finger  
    
    ```
    
    >  Print system information
    
    ```shell
         
    uname  
    
    ```
    
    >  Print the system date and time
    
    ```shell
         
    date  
    
    ```
    
    >  Print a calendar
    
    ```shell
         
    cal  
    
    ```
    
    >  Show how long the system has been running
    
    ```shell
         
    uptime  
    
    ```
    
    >  Change file access permissions
    
    ```shell
         
    chmod  
    
    ```
    
    >  Change file ownership
    
    ```shell     
    
    chown  
    
    ```
    
    >  Show the full path of a command
    
    ```shell     
    
    which  
    
    ```
    
    >  Create or extract an archive file
    
    ```shell
         
    tar  
    
    ```
    
    >  Compress or decompress files
    
    ```shell
         
    gzip  
    
    ```
    
    >  Create links between files
    
    ```shell
         
    ln  
    
    ```
    
    >  Report process status
    
    ```shell 
        
    ps  
    
    ```
    
    >  Terminate a process
    
    ```shell
         
    kill  
    
    ```
    
    >  Display system resource usage
    
    ```shell     
    
    top  
    
    ```
    
    >  Report disk usage
    
    ```shell     
    
    du  
    
    ```
    
    >  Show disk space usage
    
    ```shell
         
    df  
    
    ```
    
    >  Mount a filesystem
    
    ```shell
         
    mount  
    
    ```
    
    >  Unmount a filesystem
    
    ```shell
         
    umount  
    
    ```
    
    >  Echo a string
    
    ```shell
         
    echo  
    
    ```
    
    >  Print/set the system date and time
    
    ```shell
         
    date  
    
    ```

    >  Show manual pages
    
    ```shell     
    
    man  
    
    ```
    
    >  Show the first few lines of a file
    
    ```shell     
    
    head  
    
    ```
    
    >  Show the last few lines of a file
    
    ```shell
         
    tail  
    
    ```

    ```shell     
    
    ssh 
    
    ```
